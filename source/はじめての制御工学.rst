--------------------
2.2.1 機械系のモデル
--------------------

.. math::

   Ma(t) = f(t)

- :math:`M` : 物体の質量
- :math:`a(t)` : 時刻 :math:`t` 時点での物体の加速度
- :math:`f(t)` : 時刻 :math:`t` 時点での物体に加える力
- :math:`x(t)` :  時刻 :math:`t` 時点での物体の変位

.. math::

   a(t) = \ddot{x}(t) = \frac{d^2x(t)}{dt^2}

:math:`(1)` に :math:`(2)` を代入。

.. math::

   M\ddot{x}(t) = f(t) または M\frac{d^2x(t)}{dt^2} = f(t)
